# Student Performance Data Analysis

This project analyzes student academic data using Python.

## Libraries Used
- Pandas
- Matplotlib

## Dataset Features
- Age
- Gender
- GPA
- Study habits
- Activities

## Analysis Performed
1. Student performance classification (Good / Poor)
2. Bar graph of student performance
3. Pie chart of performance distribution
4. GPA distribution
5. Gender vs GPA analysis
6. Age distribution

## Files
- analysis.py → Python analysis code
- students.xlsx → Dataset
